# eeg-neuroscience-analyse

J’ai réalisé ce projet afin de mieux comprendre comment les signaux EEG peuvent être utilisés pour analyser l’activité cérébrale et détecter des anomalies comme les crises épileptiques.

Ce projet m’a permis de découvrir comment on peut analyser des données biologiques réelles et utiliser des outils informatiques pour étudier le fonctionnement du cerveau.L’objectif de ce projet est de distinguer des signaux EEG correspondant à une activité cérébrale normale de ceux associés à une crise épileptique.L’EEG permet d’enregistrer l’activité électrique du cerveau et est souvent utilisé pour étudier certaines pathologies neurologiques. J’ai commencé par charger les données EEG et observer leur structure.Chaque ligne correspond à un signal EEG et contient des mesures de l’activité électrique au cours du temps.J’ai ensuite séparé les données en deux groupes : un groupe pour entraîner le modèle et un autre pour tester ses performances. Le modèle apprend à reconnaître des différences entre les signaux normaux et les signaux associés aux crises. Nous observons que le modèle est capable de différencier les signaux EEG normaux des signaux de crise.

On peut en déduire que les signaux EEG contiennent des caractéristiques qui permettent de distinguer différents états de l’activité cérébrale.
Les signaux EEG reflètent l’activité électrique des neurones.

Lors d’une activité normale, les neurones fonctionnent de manière relativement stable.
Pendant une crise épileptique, les neurones présentent une activité anormale et excessive, ce qui modifie le signal EEG.
Ces différences peuvent être détectées grâce à l’analyse des données.
